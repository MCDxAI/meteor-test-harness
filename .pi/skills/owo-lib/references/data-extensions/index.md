# Data Extensions

Extended data format support for Minecraft mods: JSON5, nested language files, rich translations, and recipe remainders.

## File Index

- `json5.md` - Enabling and using JSON5 support in resource packs
- `nested-lang.md` - Nested language file support (object, array, and nested nesting patterns)
- `rich-translations.md` - Rich translation system for styled translatable text
- `recipe-remainders.md` - Recipe remainder handling with translatable text parameters
