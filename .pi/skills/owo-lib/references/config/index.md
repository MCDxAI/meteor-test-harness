# Configuration

owo-lib's configuration system with annotation-driven config classes, automatic UI generation, and sync support.

## File Index

- `getting-started.md` - Wiki tutorial: creating configs, options, sections, nested objects, ModMenu integration
- `annotations.md` - Constraint annotations (@RegexConstraint, @RangeConstraint, etc.) and screen annotations (@Hook, @Expanded, etc.)
- `synchronization.md` - Config synchronization concepts, sync modes, detached options, sync flow
- `api.md` - JavaDoc API reference for config classes
